
# Portfolio (React + TypeScript + Vite)

Minimal setup for React, TypeScript, and Vite with HMR and ESLint.

## Run Locally
```bash
npm install
npm run dev
```

## Features
- Glassmorphic UI
- Theme toggle
- Responsive
- Project showcase

## ESLint Configuration Tips
- Enable type-aware lint rules for production:
	- Set `parserOptions.project` and `tsconfigRootDir` in ESLint config.
	- Use `plugin:@typescript-eslint/recommended-type-checked` or `plugin:@typescript-eslint/strict-type-checked`.
	- Add `plugin:@typescript-eslint/stylistic-type-checked` for style.
	- Install `eslint-plugin-react` and add `plugin:react/recommended` & `plugin:react/jsx-runtime` to `extends`.



This is a modern portfolio web app built with React and Vite.
